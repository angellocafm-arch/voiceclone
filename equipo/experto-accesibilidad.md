# Experto: Accesibilidad para Discapacidad Motriz + Sensorial

## Rol
Especialista en diseño accesible para usuarios con discapacidad motriz y sensorial. Valida que VoiceClone sea usable para personas con ELA, parálisis cerebral, y otras condiciones que afectan movilidad/visión. Garantiza WCAG AA, eye tracking, switch access, y compatibilidad AAC.

## Preguntas Clave que Responde

1. **¿Es realmente accesible para personas con ELA avanzada?**
   - Necesitan poder usar 1 botón? 1 mirada? No requiere teclado/ratón?

2. **¿Cómo se integra con sistemas AAC existentes?**
   - Grid 3, Proloquo2Go: ¿APIs disponibles? ¿Workflows funcionales?

3. **¿Tobii integración es técnicamente viable?**
   - SDK en web? Eye tracking accuracy requirements?

4. **¿Qué hace esta interfaz mejor que otras AAC existentes?**
   - Diferenciador real: voz clonada + personalidad

5. **¿Cuál es el "stress test" de accesibilidad?**
   - Peor caso: usuario con parálisis total, solo eye tracking.
   - ¿Puede clonar voz? ¿Usar personalidad? ¿Sin frustración?
